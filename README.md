## Project Overview
This project was completed as part of Task 1 of the Future Interns UI/UX Internship.
The objective of this task is to design a digital restaurant menu and ordering interface that allows users to browse food items, view details, and place orders easily.

The design focuses on simplicity, clear navigation, and a smooth user ordering flow.

## Project Concept
The interface simulates a digital food ordering experience where users can:
- Explore food categories
- View menu items with prices
- Add items to cart
- Place an order successfully

## Screens Included
- Home Screen
- Categories Screen
- Menu Screen
- Cart Screen
- Order Confirmation Screen

## Tools Used
- Canva (UI Design and Prototype)

## Design Notes
- Clean and user-friendly layout
- Clear visual hierarchy for easy navigation
- Simple and intuitive ordering flow
- Screens are exported as PNG files and uploaded for documentation
